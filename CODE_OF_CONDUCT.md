# Code of Conduct

HermesShell follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

**Summary:** Be respectful. Focus on the work. Assume good intent. Disagreements about technical direction belong in issues — not personal attacks.

## Scope

This applies to all project spaces: GitHub issues, pull requests, discussions, and any community channels associated with HermesShell.

## Enforcement

Violations can be reported by opening a private issue or contacting the maintainers directly via GitHub. All reports are handled confidentially.

Maintainers reserve the right to remove comments, close issues, and ban contributors who repeatedly violate these standards.
